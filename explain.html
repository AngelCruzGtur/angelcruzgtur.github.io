How it works
Four steps from image to slime texture.
01
Load the Base64 image
The PNG is stored as a data: URL inside JavaScript — no server, no HTTP request.
02
Draw it on canvas
A hidden canvas lets the code crop any pixel region from the sprite sheet with pixel-perfect accuracy.
03
Create Three.js textures
Each cropped canvas is wrapped in a CanvasTexture with nearest-filter sampling for crisp pixels.
04
Build the slime
An outer textured cube wraps a smaller semi-transparent inner cube, giving the slime its gooey depth.

Cropping the texture
Treat the sprite sheet like a grid — copy each region to its own canvas.
JS
slime.js 
function cropTexture(x, y, w, h, outSize = 64) {
  const face = document.createElement("canvas")
  face.width  = outSize
  face.height = outSize

  const ctx = face.getContext("2d")
  ctx.imageSmoothingEnabled = false
  ctx.drawImage(source, x, y, w, h, 0, 0, outSize, outSize)

  const texture = new THREE.CanvasTexture(face)
  texture.magFilter = THREE.NearestFilter
  texture.minFilter = THREE.NearestFilter

  return texture
}
Applying materials to the cube
Each of the six faces gets its own cropped region.
JS
slime.js 
function material(x, y, w, h, opacity = 0.72) {
  return new THREE.MeshLambertMaterial({
    map:         cropTexture(x, y, w, h),
    transparent: true,
    alphaTest:   0.01,
    opacity,
    side:        THREE.DoubleSide,
  })
}

const outerCube = new THREE.Mesh(
  new THREE.BoxGeometry(4.7, 4.7, 4.7),
  [
    material( 0, 8, 8, 8),  // right
    material(16, 8, 8, 8),  // left
    material( 8, 0, 8, 8),  // top
    material(16, 0, 8, 8),  // bottom
    material( 8, 8, 8, 8),  // front
    material(24, 8, 8, 8),  // back
  ]
)
The slime effect
A semi-transparent inner cube gives the model its gooey depth.
JS
slime.js 
const innerCube = new THREE.Mesh(
  new THREE.BoxGeometry(4.7 * 0.7, 4.7 * 0.7, 4.7 * 0.7),
  new THREE.MeshLambertMaterial({
    color:       0x4fc54a,
    transparent: true,
    opacity:     0.42,
    side:        THREE.DoubleSide,
  })
)

slime.add(outerCube, innerCube)
